운영체제(OS)

    각 하드웨어에 적절한 전기신호를 흘려주는 소프트웨어이다.
    실행 가능한 프로그햄과 하드웨어를 연결해주는 역할을 한다.

Linux®(리눅스)

    Linux®(이하 Linux)에서의 ®은 registered trademark symbol로서, 등록된 상표임을 알리기 위해 사용한다.
    Linux는 오픈소스(Open Soure Software, OSS) 운영체제(OS)이다.
    이 때, 오픈소스는 공개적으로 접근할 수 있게 설계되어 누구나 자유롭게 확인, 수정, 배포할 수 있는 코드를 의미한다.

오픈소스(Open Source)

    Linux GNU GPL(GNU General Public License)를 사용한다.
    GNU GPL이란, 자유 소프트웨어 재단에서 만든 자유 소프트웨어 라이선스를 의미한다.
    즉, 누구든지 소스코드를 실행, 연구, 수정, 재배포할 수 있다.
    동일한 라이선스가 유지되는 한 수정힌 코드의 복사본을 판매할 수도 있다.
